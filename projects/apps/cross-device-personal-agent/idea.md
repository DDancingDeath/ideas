# Cross-Device Personal Agent

> A personal agent system that bridges my phone and my development machines: a cloud control plane routes commands from my phone to the right dev box, a small device agent on each machine executes the action (launch app, open file, run script, query status), and results plus long-running task notifications are pushed back. Starts as a plain remote command runner and evolves into a natural-language, multi-agent assistant spanning all my devices.
